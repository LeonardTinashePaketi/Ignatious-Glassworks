## Recommended Image Sources

1. **Your Own Photos** — Best option. Use high-quality photos of your completed installations.
2. **Client Approvals** — With permission, use photos from satisfied clients.
3. **On-Site Shoots** — Schedule professional photography of your best work.

## Tips for Best Results

- Use high-resolution images (at least 2x the display size)
- Ensure images are properly lit — glass photography needs good lighting
- For glass work, include context shots showing the surrounding architecture
- Portfolio images should showcase variety across your services
- Use consistent editing style across all images for cohesive look

## Quick Start

1. Replace each `.jpg` file in this folder with your own image
2. Keep the same filename to maintain all references
3. If you rename files, update the paths in `index.html`
